brigadeiros;
clientes;
funcionarios.

requisitos funcionais:

cadastro, login e senha
busca
"fazer carrinho de compra"
realizar pedido 
cancelar pedido
rever pedido

requisitos não funcionais

notificar cliente caso não seja aceito
